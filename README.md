## Client Server Model for multiplayer Tic-Tac-Toe Game in Local Area Network ##

#### Step-1: ####
> Start Server

> Run TicTacToeServer.java

#### Step-2: ####
> Start Client for player-1

> Run TicTacToeCleint.java

#### Step-3: ####
> Start Client for player-2

> Run TicTacToeCleint.java

#### Step-4 ####
> Enjoy
